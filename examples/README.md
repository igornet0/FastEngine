# Примеры использования FastEngine

Этот каталог содержит примеры использования FastEngine для различных сценариев.

## Базовый пример

### main.cpp
Простейший пример создания окна и игрового цикла.

### Game.cpp/Game.h
Пример игрового класса с базовой логикой.

## Запуск примеров

### Desktop
```bash
# Сборка
./scripts/build_desktop.sh

# Запуск
./build/linux/ExampleApp
# или
./build/macos/ExampleApp
# или
./build/windows/ExampleApp.exe
```

### Android
```bash
# Сборка
./scripts/build_android.sh

# Установка на устройство
adb install build/android/ExampleApp.apk
```

### iOS
```bash
# Сборка
./scripts/build_ios.sh

# Откройте проект в Xcode для установки на устройство
```

## Структура примеров

- `main.cpp` - точка входа в приложение
- `Game.h/Game.cpp` - основной игровой класс
- `README.md` - этот файл с инструкциями

## Добавление новых примеров

1. Создайте новую директорию в `examples/`
2. Добавьте `CMakeLists.txt` для сборки
3. Создайте исходные файлы
4. Обновите основной `CMakeLists.txt` для включения нового примера
5. Добавьте описание в этот README

## Примеры по категориям

### Базовые
- **Hello World** - простейший пример

## Советы по разработке

1. **Начинайте с простого** - сначала изучите базовые примеры
2. **Изучайте ECS** - это основа архитектуры движка
3. **Используйте компоненты** - не наследуйтесь от Entity
4. **Пишите системы** - логика должна быть в системах
5. **Тестируйте на разных платформах** - убедитесь, что код работает везде
