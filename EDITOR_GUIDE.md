# Руководство по использованию Simple Game Editor

## Обзор

Simple Game Editor - это простой визуальный редактор для создания 2D игр на движке FastEngine. Редактор позволяет создавать сцены, размещать объекты, настраивать их свойства и сохранять проекты.

## Установка и запуск

### Требования
- macOS 10.14+ (или Linux/Windows с соответствующими зависимостями)
- CMake 3.16+
- C++17 компилятор
- OpenGL 3.3+
- SDL2
- jsoncpp

### Сборка
```bash
cd /Users/igor/Desktop/GameAppsAi/FastEngine
mkdir -p build_editor
cd build_editor
cmake -f ../CMakeLists_editor.txt -DCMAKE_BUILD_TYPE=Release
make -j4
```

### Запуск
```bash
./SimpleGameEditor
```

## Управление

### Основные клавиши

| Клавиша | Действие |
|---------|----------|
| **ESC** | Выход из редактора |
| **Space** | Переключение отображения сетки |
| **G** | Переключение привязки к сетке |
| **I** | Переключение панели инспектора |
| **H** | Переключение панели иерархии |
| **A** | Переключение браузера ассетов |
| **S** | Сохранение проекта |
| **L** | Загрузка проекта |
| **N** | Создание нового проекта |

### Управление камерой

| Клавиша | Действие |
|---------|----------|
| **W** | Движение камеры вверх |
| **S** | Движение камеры вниз |
| **A** | Движение камеры влево |
| **D** | Движение камеры вправо |

### Управление объектами

| Клавиша | Действие |
|---------|----------|
| **Стрелки** | Движение выбранного объекта |
| **Касание/Клик** | Выбор объекта |
| **Двойное касание** | Создание нового объекта |

## Интерфейс

### Панели

1. **Сетка** - Визуальная сетка для точного размещения объектов
2. **Инспектор** - Панель свойств выбранного объекта
3. **Иерархия** - Список всех объектов в сцене
4. **Браузер ассетов** - Просмотр доступных ресурсов

### Состояния интерфейса

Редактор отображает текущее состояние интерфейса в консоли:
- `Grid: ON/OFF` - Состояние сетки
- `Snap: ON/OFF` - Состояние привязки к сетке
- `UI - Inspector: ON/OFF` - Состояние панели инспектора
- `UI - Hierarchy: ON/OFF` - Состояние панели иерархии
- `UI - Assets: ON/OFF` - Состояние браузера ассетов

## Работа с проектами

### Создание нового проекта
1. Нажмите клавишу **N**
2. Все существующие объекты будут удалены
3. Создается пустая сцена

### Сохранение проекта
1. Нажмите клавишу **S**
2. Проект сохраняется в файл `my_game.json`
3. В консоли появится сообщение "Project saved to my_game.json"

### Загрузка проекта
1. Нажмите клавишу **L**
2. Загружается проект из файла `my_game.json`
3. Если файл не найден, создается пустая сцена

## Работа с объектами

### Создание объектов
1. Используйте двойное касание или два пальца на сенсорном экране
2. Объект создается в позиции касания
3. Объект автоматически выбирается после создания

### Выбор объектов
1. Кликните или коснитесь объекта
2. Выбранный объект подсвечивается
3. В консоли отображается позиция выбранного объекта

### Перемещение объектов
1. Выберите объект
2. Используйте стрелки для перемещения
3. Если включена привязка к сетке, объект будет привязываться к узлам сетки

## Формат проекта

Проекты сохраняются в формате JSON:

```json
{
  "name": "My Game",
  "version": "1.0.0",
  "entities": [
    {
      "x": 100.0,
      "y": 200.0,
      "type": "sprite"
    }
  ]
}
```

### Структура файла проекта

- `name` - Название проекта
- `version` - Версия проекта
- `entities` - Массив объектов в сцене
  - `x`, `y` - Позиция объекта
  - `type` - Тип объекта (sprite)

## Настройки

### Размер сетки
По умолчанию размер сетки составляет 32 пикселя. Это значение можно изменить в коде редактора.

### Скорость движения
- Камера: 300 пикселей в секунду
- Объекты: 200 пикселей в секунду

## Примеры использования

### Создание простой сцены
1. Запустите редактор
2. Нажмите **N** для создания нового проекта
3. Используйте двойное касание для создания объектов
4. Перемещайте объекты стрелками
5. Нажмите **S** для сохранения

### Редактирование существующей сцены
1. Запустите редактор
2. Нажмите **L** для загрузки проекта
3. Выберите объекты кликом
4. Перемещайте их стрелками
5. Нажмите **S** для сохранения изменений

## Ограничения

- Редактор работает только в консольном режиме (без графического интерфейса)
- Поддерживается только один тип объектов (sprite)
- Нет возможности удаления объектов
- Нет возможности изменения свойств объектов (размер, цвет, текстура)

## Разработка

### Структура кода
- `SimpleGameEditor` - Основной класс редактора
- `Initialize()` - Инициализация редактора
- `Update()` - Обновление логики
- `Render()` - Отрисовка интерфейса
- `HandleInput()` - Обработка ввода
- `SaveProject()` - Сохранение проекта
- `LoadProject()` - Загрузка проекта

### Добавление новых функций
1. Добавьте новые клавиши в `HandleInput()`
2. Реализуйте логику в соответствующих методах
3. Обновите документацию

## Поддержка

При возникновении проблем:
1. Проверьте, что все зависимости установлены
2. Убедитесь, что движок FastEngine собран
3. Проверьте права доступа к файлам проекта
4. Посмотрите сообщения в консоли

## Заключение

Simple Game Editor предоставляет базовый функционал для создания 2D игр. Хотя он имеет ограничения, он служит хорошей основой для дальнейшего развития и может быть расширен дополнительными функциями по мере необходимости.
